## Front

<br/>

<ol>
    <li>Instal "Node.js" from <a href="https://nodejs.org/">https://nodejs.org</a>.</li>
    <li>Run "bash npm install" (or "npm i" in Windows shell/console) from project directory</li>
</ol>

* ```npm run dev``` - development server on localhost:8000
* ```npm run production``` - create production version to ./prod
